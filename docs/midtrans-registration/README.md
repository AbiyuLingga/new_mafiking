# Dokumen Registrasi Midtrans Mafiking

Folder ini berisi dokumen pendukung untuk menjawab permintaan Midtrans:

> Informasi end to end process dari customer memesan produk barang/jasa sampai customer memproses pembayaran.

## Isi Folder

- `end-to-end-process-mafiking.md` - dokumen utama yang bisa diunggah sebagai lampiran.
- `end-to-end-process-copy-paste.txt` - versi ringkas yang bisa ditempel ke form atau email Midtrans.
- `submission-checklist.md` - checklist sebelum mengirim ulang dokumen ke Midtrans.

## Catatan Akurasi

Dokumen ini menjelaskan alur bisnis dan pembayaran Mafiking untuk kebutuhan review merchant. Kode aplikasi saat ini memiliki payment flow berbasis payment gateway dan akan disesuaikan ke Midtrans setelah akun produksi aktif. Hindari menulis bahwa transaksi Midtrans sudah live sebelum konfigurasi production Midtrans benar-benar aktif.
